# risk
grade 12 project, created a game of risk using graphs


### INTRO ###

This project was created using processing3 and Python.

The project was done in combination with a presentation on graph theory in grade 12 as an fun additional project.

To learn how to play click on the howTo.txt file in https://github.com/JackDemeter/risk/tree/master/Game/gameProcessing/data

The program uses classes to create nodes that 'point'(no pointers in python) to other node objects to form a graph. By selecting a node with your mouse the program searches all neighbours of said node and highlights enemys that are attackable, similar concept for moving troops between friendly nodes.
